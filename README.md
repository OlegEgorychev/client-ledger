# Client Ledger - Android приложение для учёта клиентов, записей и расходов

Полнофункциональное Android приложение на Kotlin для ведения учёта клиентов, записи доходов и расходов с календарём и статистикой.

## Технологии

- **Kotlin** - язык программирования
- **Jetpack Compose** - современный UI toolkit
- **Material 3** - дизайн система
- **Room Database** - локальная база данных SQLite
- **MVVM** - архитектурный паттерн
- **Coroutines + Flow** - асинхронное программирование
- **Navigation Compose** - навигация между экранами

## Минимальные требования

- **Минимальная версия Android**: 26 (Android 8.0)
- **Target SDK**: 34
- **Android Studio**: Arctic Fox или новее

## Как запустить

1. **Откройте проект в Android Studio**
   - Откройте Android Studio
   - Выберите "Open an Existing Project"
   - Выберите папку `D:\Programs\test`

2. **Синхронизация Gradle**
   - Android Studio автоматически начнёт синхронизацию Gradle
   - Дождитесь завершения синхронизации

3. **Подключите устройство или эмулятор**
   - Подключите Android устройство через USB с включённой отладкой
   - Или запустите эмулятор Android (API 26 или выше)

4. **Запустите приложение**
   - Нажмите кнопку "Run" (Shift+F10) или выберите Run > Run 'app'
   - Выберите устройство/эмулятор
   - Приложение установится и запустится

## Функциональность

### Вкладка "Клиенты"
- Просмотр списка всех клиентов
- Поиск по имени, фамилии, телефону, Telegram
- Добавление нового клиента
- Редактирование существующего клиента
- Просмотр деталей клиента
- Удаление клиента (с подтверждением)

### Вкладка "Календарь"
- Календарная сетка текущего месяца
- Навигация по месяцам (вперёд/назад)
- Выбор даты для просмотра деталей дня
- Просмотр записей клиентов (доходы) за день
- Просмотр расходов за день
- Добавление записи клиента и расходов

### Вкладка "Статистика"
- Выбор периода: День, Месяц, Год
- Просмотр статистики: доход, расход, прибыль, рабочие дни

## Особенности

- **Валюта**: все суммы хранятся в копейках (Long), отображаются в рублях
- **Демо-данные**: при первом запуске автоматически создаются демо-клиенты, записи и расходы
- **Язык интерфейса**: русский
- **Календарь**: реализован без внешних библиотек
